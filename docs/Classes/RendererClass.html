RendererClass

open class RendererClass : RendererClassProtocol

The RendererClass type acts as an owner of an underlying CoglPangoRendererClass instance. It provides the methods that can operate on this data type through RendererClassProtocol conformance. Use RendererClass as a strong reference or owner of a CoglPangoRendererClass instance.
